== Compiling
* Arch Linux (also supports WSL2):
. `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
. `source ~/.bashrc`
. `sudo pacman -Syu`
. `sudo pacman -S base-devel gdb clang cmake ninja mold util-linux-libs parted`
